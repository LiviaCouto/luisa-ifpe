# Luísa IFPE 2027 🌟

App de preparação personalizado para o processo seletivo do **IFPE 2027** — Técnico Integrado.

## Acesso online

[Abrir app](https://liviacouto.github.io/luisa-ifpe)

## Sobre o projeto

Plataforma de estudos desenvolvida com amor por uma mãe para a filha. Combina plano de estudos estruturado, gamificação e acompanhamento de progresso em tempo real — tudo baseado no edital oficial do IFPE.

## Funcionalidades

- 📅 Plano de estudos dia a dia até 20/12 baseado no edital
- 🔥 Streak de dias consecutivos estudando
- ✦ Sistema de conquistas desbloqueáveis
- 📊 Dashboard com analytics, metas e gráficos
- 🔁 Revisão espaçada automática (24h, 7d, 30d)
- ⏱ Cronômetro de sessão integrado
- 🎯 Registro de acertos e erros por matéria
- 🌙 Dark e light mode
- 📱 Responsivo para celular
